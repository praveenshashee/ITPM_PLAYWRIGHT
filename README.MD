# Singlish to Sinhala Translator – Playwright Test Automation

## Module
ITPM – Software Project Management

## Description
This project contains automated functional and UI test cases for the Singlish to Sinhala transliteration system available at:

https://www.swifttranslator.com/

The tests are written using Microsoft Playwright and focus on:
- Functional correctness of Singlish to Sinhala conversion
- Real-time output generation behavior
- Robustness against mixed language and informal inputs
- Basic UI-related behaviors

Backend APIs, performance, and security testing are not included as per assignment requirements.
